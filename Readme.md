# benford
Implemantion of benford algorithm

# Building
```shell
make build
```

# Testing
```shell
make test
```

# Running Benford
```shell
./bin/benford [-csv string] [-colIndex int]  [-version]

-colIndex : Index of a column which is to be processed (default: 5)
-csv : csv file (default: "test.csv")
```
